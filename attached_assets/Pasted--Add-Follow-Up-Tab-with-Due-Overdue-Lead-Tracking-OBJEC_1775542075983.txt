# ✅ STEP 3: FILTER LOGIC (CORE FEATURE)

---

## 🧠 Define Dates

```ts id="b1ep0k"
const today = new Date();
today.setHours(0, 0, 0, 0);
```

---

## 🔹 Filter Leads

```ts id="p6r3hz"
const followUpLeads = leads.filter(lead => {
  if (!lead.followUpDate) return false;

  const followDate = new Date(lead.followUpDate);
  followDate.setHours(0, 0, 0, 0);

  return (
    followDate.getTime() === today.getTime() ||
    followDate < today
  );
});
```

---

## ✔ Result

* Shows:

  * Today’s follow-ups
  * Overdue leads

# ✅ STEP 4: OVERDUE CALCULATION

---

## 🔹 Calculate Days Overdue

```ts id="1ct2wq"
const getOverdueDays = (date) => {
  const today = new Date();
  const follow = new Date(date);

  const diffTime = today - follow;
  return Math.floor(diffTime / (1000 * 60 * 60 * 24));
};
```
