## 🚀 Replace "Created Date" with "Last Updated Date" in Leads Table

---

# 🎯 OBJECTIVE

Update the Leads Table to:

* Replace **"Created" column → "Updated On" (Last Updated Date)**
* Sort leads by **most recently updated first**
* Automatically update timestamp on every change
* Ensure **ZERO data loss / ZERO data mutation**

---

# ⚠️ CRITICAL RULES

* ❌ Do NOT modify existing lead data
* ❌ Do NOT delete or overwrite `createdAt`
* ✅ Only **add/use `updatedAt` field**
* ✅ UI-only + safe backend update tracking

---

# ✅ STEP 1: ENSURE `updatedAt` FIELD EXISTS

---

## 🧠 Backend Logic

If not already present, ensure:

```ts
updatedAt: timestamp (auto-managed)
```

---

## ✔ Auto-update on every change

```ts
await db.leads.update({
  where: { id: leadId },
  data: {
    ...updatedFields,
    updatedAt: new Date()
  }
});
```

---

## ✔ IMPORTANT

* Applies to:

  * Lead edits
  * Status updates
  * Notes added
  * Owner/handler change
  * Service/company updates
  * Any changes made to the Lead

---

---

## 🔹 Replace Column Header

```tsx

```

---

## 🔹 Replace Data Binding

```tsx

</td>
```

---

## ✔ Fallback Logic

* If `updatedAt` is missing → show `createdAt`

---

# ✅ STEP 3: SORT BY LAST UPDATED (DEFAULT)

---

## 🔹 Backend Query (Preferred)

```ts
orderBy: {
  updatedAt: 'desc'
}
```

---

## 🔹 OR Frontend Sorting

```ts
leads.sort((a, b) => {
  return new Date(b.updatedAt || b.createdAt) - new Date(a.updatedAt || a.createdAt);
});
```

---

## ✔ RESULT

* Most recently updated lead → top
* Older leads → below
