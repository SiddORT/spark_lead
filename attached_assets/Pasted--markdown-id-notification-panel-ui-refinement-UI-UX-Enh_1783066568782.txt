# UI/UX Enhancement – Refine Notification Panel Layout, Readability & Visual Design

## Objective

The Notification Panel is currently suffering from multiple UI and UX issues that make it look unfinished and inconsistent with the premium quality of the SparkLead application.

As visible in the attached screenshot:

- Text is overlapping and unreadable.
- User information is colliding with lead titles and dates.
- Icons are outdated and visually inconsistent.
- Spacing and alignment are poor.
- The overall notification panel does not match the design language of the rest of the application.

This entire component needs to be redesigned and refined while preserving all existing functionality.

**Important:** This is a UI/UX refinement only. Do not modify the notification logic, APIs, database, permissions, or business workflows.

---

# Current Issues

The notification panel currently has the following problems:

- Lead titles overlap with other text.
- User names are rendered on top of dates.
- Long lead names break the layout.
- Text hierarchy is unclear.
- Notification cards have inconsistent spacing.
- Icons look outdated and do not match the rest of the application.
- Poor alignment between text, badges, and action areas.
- Overall appearance feels cluttered and difficult to scan.

---

# Required UI Improvements

Redesign each notification item using a clean card/list layout.

Each notification should clearly display:

- Lead Name
- Notification Type
- Assigned User / Triggered By
- Due Date or Activity Date
- Status Badge (Overdue, Today, Upcoming, Completed, etc.)
- Optional Lead Stage Badge (if applicable)

Each piece of information should occupy its own dedicated line or section to prevent any overlap.

Example layout:

```
📌 Maharashtra Solar Project

👤 Smit Lalai

📅 Wed, 10 Jun 2026

⚠️ Overdue
```

Everything should remain perfectly aligned regardless of content length.

---

# Typography Improvements

Improve text hierarchy by using:

### Lead Name

- Bold
- Largest text
- Single or maximum two lines
- Apply ellipsis (...) if extremely long

---

### User Name

- Medium weight
- Secondary text color

---
