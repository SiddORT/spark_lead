## ✅ 3. Calculate Stage Counts

```ts
const stageCounts = stages.map((stage) => {
  const count = dataSource.filter((lead) => lead.stage === stage).length;

  return {
    stage,
    count,
  };
});
```

---

## ✅ 4. Calculate Total Leads

```ts
const totalLeads = dataSource.length;
```

---

## ✅ 5. Calculate Percentages

```ts
const stageData = stageCounts.map((item) => ({
  ...item,
  percentage: totalLeads > 0 ? (item.count / totalLeads) * 100 : 0,
}));
```

---

## ✅ 6. Render Proper Graph (Progress Bars)

```tsx
{stageData.map((item) => (
  <div key={item.stage} className="mb-4">
    
    <div className="flex justify-between text-sm text-gray-400 mb-1">
      <span>{item.stage}</span>
      <span>
        {item.count} {item.count === 1 ? "lead" : "leads"} ({Math.round(item.percentage)}%)
      </span>
    </div>

    <div className="w-full h-2 bg-gray-800 rounded-full overflow-hidden">
      <div
        className="h-full bg-teal-500 transition-all duration-500"
        style={{ width: `${item.percentage}%` }}
      />
    </div>

  </div>
))}
```
