## 🚀 Add “Create Service On-The-Fly + Link to Company” in Create Lead Form

---

# 🎯 Objective

Enhance **Create New Lead** form to allow:

### ✅ User can:

* Add a **new Service directly from dropdown**
* Link that service to one or more **Companies**
* Automatically:

  * Select the newly created service in form
  * Update company dropdown accordingly
* Continue lead creation without losing form state

---

# ❗ Problem Today

* Service must exist beforehand
* Breaks flow if missing
* No quick way to add + link service

---

# ✅ EXPECTED UX FLOW

1. User opens **Service dropdown**
2. If service not found → clicks:

👉 **“+ Add New Service”**

3. Modal opens:

   * Enter Service Name
   * Select Companies to link

4. Click **Save**

5. System:

   * Saves new service
   * Links it with selected companies
   * Auto-selects it in form
   * Updates Company dropdown

## ✅ 4. Backend: Create Service + Link Companies

### DB Structure:

```sql
services (id, name)
companies (id, name)
service_company_map (service_id, company_id)
```

---

### API: `/api/create-service`

```ts id="svc4"
export async function createService(data) {
  const { name, companyIds } = data;

  // Insert service
  const { data: service } = await supabase
    .from("services")
    .insert({ name })
    .select()
    .single();

  // Link companies
  const mappings = companyIds.map((id) => ({
    service_id: service.id,
    company_id: id,
  }));

  await supabase.from("service_company_map").insert(mappings);

  return service;
}
```

## ✅ 5. Handle Save in Frontend

```ts id="svc5"
const handleCreateService = async () => {
  const newService = await createService({
    name: newServiceName,
    companyIds: selectedCompanies,
  });

  // Update services list
  setServices((prev) => [...prev, newService]);

  // Auto select new service
  setSelectedService(newService.name);

  // Update company dropdown
  setSelectedCompany("");

  setShowServiceModal(false);
};
```
