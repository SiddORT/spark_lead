# 🚀 FEATURE: FLOATING “+” BUTTON FOR QUICK LEAD CREATION

---

# 🎯 OBJECTIVE

Add a **Floating Action Button (FAB)** at the **bottom-right corner** of the application.

When clicked:
✅ It should open the **Create New Lead form**
✅ Same form used in “New Lead” page
✅ Should work from **any page (global access)**

---

# 🧩 FUNCTIONAL REQUIREMENTS

---

## 🔹 BUTTON BEHAVIOUR

* Floating button visible across all pages
* Positioned at:

  ```css
  bottom: 24px;
  right: 24px;
  ```
* On click:
  → Open **Create Lead Drawer / Modal**

---

## 🔹 FORM BEHAVIOUR

* Open existing **Create Lead form**
* Do NOT create a new form
* Reuse:

  * Validation
  * Submission logic
  * API calls

---

## 🔹 AFTER SUBMIT

* Lead should be created successfully
* Drawer/Modal should close
* Dashboard / table should refresh automatically

---

# 🏗️ IMPLEMENTATION

---

## 🔹 STEP 1: ADD FLOATING BUTTON COMPONENT

```tsx
// FloatingActionButton.tsx

<button
  onClick={openCreateLeadModal}
  className="fixed bottom-6 right-6 z-50 bg-primary text-white rounded-full w-14 h-14 flex items-center justify-center shadow-lg hover:scale-105 transition"
>
  +
</button>
```

---

## 🔹 STEP 2: GLOBAL INTEGRATION

Add FAB in:

```tsx
// Layout / App shell (global layout file)
```

## 🔹 STEP 3: OPEN CREATE LEAD FORM

Use existing modal/drawer logic:

