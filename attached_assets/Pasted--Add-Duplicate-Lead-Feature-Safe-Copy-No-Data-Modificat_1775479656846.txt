## 🚀 Add “Duplicate Lead” Feature (Safe Copy – No Data Modification)

---

# 🎯 Objective

Add a **Duplicate Lead button** in the Lead Details page that allows:

### ✅ User Flow

1. Open any Lead Details page
2. Click **Duplicate (📄 icon)** beside Delete button
3. Redirect to **Create Lead form**
4. Form is **pre-filled with selected lead data**
5. User can:

   * Edit fields if needed
   * Click “Create Lead”
6. A **NEW lead is created**

---

# ❗ CRITICAL REQUIREMENTS

### 🔒 Data Safety (MOST IMPORTANT)

* ❌ Original lead MUST NOT be modified
* ❌ No overwrite of existing data
* ❌ No accidental updates
* ✅ Only CREATE new record

---

# 🧠 IMPLEMENTATION

---

## ✅ 1. Add Duplicate Button (UI)

Place beside delete button:

```tsx id="dup1"
<button
  onClick={handleDuplicate}
  className="p-2 rounded-lg hover:bg-white/10"
  title="Duplicate Lead"
>
  📄
</button>
```

---

## ✅ 2. Handle Duplicate Click

```ts id="dup2"
const handleDuplicate = () => {
  navigate("/new-lead", {
    state: {
      duplicateLead: currentLead,
    },
  });
};
```

---

## ✅ 3. Populate Create Lead Form

In Create Lead page:

```ts id="dup3"
const location = useLocation();
const duplicateLead = location.state?.duplicateLead;
```

---

## ✅ 4. Pre-fill Form Fields

```ts id="dup4"
useEffect(() => {
  if (duplicateLead) {
    setFormData({
      lead_name: duplicateLead.lead_name,
      company_name: duplicateLead.company_name,
      type: duplicateLead.type,
      value: duplicateLead.value,
      stage: duplicateLead.stage,
      status: duplicateLead.status,
      email: duplicateLead.email,
      phone: duplicateLead.phone,
      service: duplicateLead.service,
      owner_id: duplicateLead.owner_id,
      handler_id: duplicateLead.handler_id,
      notes: duplicateLead.notes,
    });
  }
}, [duplicateLead]);
```

---

## ⚠️ IMPORTANT: EXCLUDE THESE FIELDS

DO NOT copy:

```ts id="dup5"
id ❌
created_at ❌
updated_at ❌
```

---

## ✅ 5. Ensure New Record Creation

```ts id="dup6"
await supabase.from("leads").insert({
  ...formData,
});
```

👉 Must NOT use update()

## ✅ 6. Clear Navigation State (Prevent Reuse Bug)
