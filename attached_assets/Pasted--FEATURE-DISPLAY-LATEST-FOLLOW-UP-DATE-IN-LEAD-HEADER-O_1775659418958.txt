# 🚀 FEATURE: DISPLAY LATEST FOLLOW-UP DATE IN LEAD HEADER

---

# 🎯 OBJECTIVE

Display the **Latest Follow-up Date** in the **Lead Details header section**, alongside:

* Lead Type (Hot/Warm/Cold)
* Deal Value (₹)

This should reflect the **latest follow-up date selected via Notes**, not the legacy field.

---

Add the Follow-up date in the top header section:


# 🧠 DATA SOURCE LOGIC

---

## ✅ SOURCE OF TRUTH

Use:

```ts
getActiveFollowUpDate(lead)
```

---

## 🔹 Helper Function (if not already implemented)

```ts id="fup-logic"
function getActiveFollowUpDate(lead) {
  const notesWithDates = lead.notes
    .filter(n => n.followUpDate)
    .sort((a, b) => new Date(b.followUpDate) - new Date(a.followUpDate));

  if (notesWithDates.length > 0) {
    return notesWithDates[0].followUpDate;
  }

  return lead.followUpDate || null; // fallback (legacy safety)
}
```
