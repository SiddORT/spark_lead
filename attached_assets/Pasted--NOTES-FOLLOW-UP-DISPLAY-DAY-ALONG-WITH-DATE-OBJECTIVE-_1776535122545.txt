# 🧠 IMPLEMENTATION STRATEGY

---

## ✅ STEP 1: DATE FORMAT UTILITY FUNCTION

Create a reusable formatter:

```js
function formatFullDate(date) {
  return new Intl.DateTimeFormat('en-IN', {
    weekday: 'long',
    day: '2-digit',
    month: 'short',
    year: 'numeric'
  }).format(new Date(date));
}
```
