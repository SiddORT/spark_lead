# 🟢 FEATURE: WON CLOSURE ENHANCEMENTS (VALUE CHANGE + NOTES + DOCUMENTS)

---

# 🎯 OBJECTIVE

When a lead is marked:

* Stage = **Closure**
* Status = **Closed - Won**

Then enable:

1. ✏️ Editable Lead Value (with audit tracking)
2. 📝 Closure Note (mandatory optional)
3. 📂 Document Upload
4. 📊 Value Change Tracking (original vs final)
5. 🗂 New "Lead Documents" Tab
6. 🔍 Full visibility of document history

---

# 🧠 2. VALUE EDIT LOGIC (CRITICAL)

---

## WHEN:

```ts
stage === "Closure" && status === "Closed - Won"
```

---

## ENABLE:

Editable value field

---

## UI FIELD

```jsx
<input
  type="number"
  value={finalValue || originalValue}
  onChange={(e) => setFinalValue(e.target.value)}
  className="input"
/>
```

---

## SAVE LOGIC

```ts
if (finalValue && finalValue !== originalValue) {
  saveToValueHistory({
    oldValue: originalValue,
    newValue: finalValue,
    reason: "Won Adjustment"
  });

  updateLead({
    finalValue,
    valueUpdatedAt: now(),
    valueUpdatedBy: currentUser
  });
}
```
