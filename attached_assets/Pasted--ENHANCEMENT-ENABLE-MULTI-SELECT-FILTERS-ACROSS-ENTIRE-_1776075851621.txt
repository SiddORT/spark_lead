# 🔹 2. DATA STRUCTURE (FRONTEND)

---

## Store selected values as arrays:

```ts
filters = {
  services: ["CRM Implementation", "Data Analytics"],
  companies: ["TechNova Solutions", "FinEdge Partners"],
  types: ["Hot", "Warm"],
  stages: ["Qualification & Analysis", "Closure"]
};
```

---

# 🔹 3. FILTER LOGIC (VERY IMPORTANT)

---

## Apply logic as:

### BETWEEN FILTERS → AND condition

### WITHIN FILTER → OR condition

---

## Example:

```ts
filteredLeads = leads.filter(lead =>
  (filters.services.length === 0 || filters.services.includes(lead.service)) &&
  (filters.companies.length === 0 || filters.companies.includes(lead.company)) &&
  (filters.types.length === 0 || filters.types.includes(lead.type)) &&
  (filters.stages.length === 0 || filters.stages.includes(lead.stage))
);
```

---

## ✔ Expected Behaviour:

| Filters Applied | Result                               |
| --------------- | ------------------------------------ |
| 2 services      | Leads matching ANY of those services |
| 2 stages        | Leads matching ANY of those stages   |
| Service + Stage | Leads matching BOTH conditions       |
