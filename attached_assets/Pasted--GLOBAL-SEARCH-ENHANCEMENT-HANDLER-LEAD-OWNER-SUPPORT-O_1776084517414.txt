# 🔍 GLOBAL SEARCH ENHANCEMENT (HANDLER + LEAD OWNER SUPPORT)

---

# 🎯 OBJECTIVE

Enhance the existing **search bar (Search leads or companies…)** so that users can search leads using:

* Lead Name ✅ (already working)
* Company Name ✅ (already working)
* **Handler Name** 🆕
* **Lead Owner Name** 🆕

👉 Even if Handler / Owner is NOT visible in the table, search should still work.

---

# 🧠 EXPECTED BEHAVIOUR

---

## ✅ SEARCH SHOULD MATCH AGAINST:

```js
[
  lead.name,
  lead.companyName,
  lead.handlerName,
  lead.ownerName
]
```

---

## ✅ SEARCH SHOULD BE:

* Case insensitive
* Partial match supported
  ("chi" → chirag)
* Trimmed input
* Real-time filtering

---

# 🔧 BACKEND / DATA LOGIC

---

## 1. ENSURE DATA AVAILABILITY

Each lead object must include:

```json
{
  "leadName": "RVNL Dashboard Project",
  "companyName": "OneRoof Technologies",
  "handlerName": "Chirag",
  "ownerName": "Admin"
}
```

👉 If handler/owner is stored as ID → map to name before filtering

---

## 2. SEARCH FILTER FUNCTION

```js
const searchText = input.toLowerCase().trim();

const filteredLeads = leads.filter((lead) => {
  return [
    lead.leadName,
    lead.companyName,
    lead.handlerName,
    lead.ownerName
  ]
    .filter(Boolean)
    .some(field =>
      field.toLowerCase().includes(searchText)
    );
});
```
