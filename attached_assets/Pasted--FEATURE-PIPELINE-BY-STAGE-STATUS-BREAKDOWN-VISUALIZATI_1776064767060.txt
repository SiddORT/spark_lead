# 🚀 FEATURE: PIPELINE BY STAGE – STATUS BREAKDOWN VISUALIZATION

---

# 🎯 OBJECTIVE

Enhance the **“Pipeline by Stage”** dashboard chart to:

✅ Show **status-level breakdown within each stage**
✅ Display **lead count + total deal value per status**
✅ Improve hover insights for better decision-making

---

# 🧩 CURRENT BEHAVIOUR

* Chart displays:

  * Total leads per **Stage**
* Missing:

  * Status-level visibility inside each stage
  * Value (₹) insights per stage/status

---

# 🆕 REQUIRED ENHANCEMENT

---

## 🔹 1. STACKED BAR CHART (MANDATORY)

Convert existing chart into:

### 👉 **Stacked Bar Chart**

Each **Stage bar → split into multiple sections (Statuses)**

Example:

| Stage         | Status Breakdown                                |
| ------------- | ----------------------------------------------- |
| Qualification | Internal Discussion, Requirement Analysis, etc. |
| Proposal      | Proposal Shared, Negotiation, etc.              |

---

## 🔹 2. DATA STRUCTURE

Backend/API should return:

```json
[
  {
    "stage": "Qualification & Analysis",
    "statuses": [
      {
        "status": "Internal Discussion",
        "count": 10,
        "value": 2500000
      },
      {
        "status": "Requirement Analysis",
        "count": 8,
        "value": 1800000
      }
    ]
  }
]
```

---

## 🔹 3. CHART IMPLEMENTATION

If using Recharts / Chart.js:

### Example (Recharts):

```tsx
<BarChart data={data}>
  <XAxis dataKey="stage" />
  <YAxis />
  <Tooltip content={<CustomTooltip />} />
  
  {allStatuses.map((status) => (
    <Bar
      key={status}
      dataKey={status}
      stackId="a"
      fill={statusColorMap[status]}
    />
  ))}
</BarChart>
```

## 🔹 5. COLOR MAPPING

* Each **status → unique color**
* Should remain consistent across system

Example:

```ts
const statusColorMap = {

  "Requirement Analysis": "#10B981",
  "Proposal Shared": "#6366F1",
};
```

---

## 🔹 6. DATA TRANSFORMATION

Transform backend data into chart format:

```ts
const formattedData = stages.map(stage => {
  const obj = { stage: stage.name };

  stage.statuses.forEach(status => {
    obj[status.name] = status.count;
  });

  return obj;
});
```
