# FEATURE OVERVIEW

Users should now be able to:

✅ Follow specific leads  
✅ Follow multiple leads  
✅ Receive updates for followed leads  
✅ Unfollow leads  
✅ View followed leads list  
✅ Get notifications for followed leads

Users who are already:
- Lead Owner

# USER CAN FOLLOW LEADS ONLY IF

The user is:

❌ NOT Lead Owner

❌ NOT Lead Handler
