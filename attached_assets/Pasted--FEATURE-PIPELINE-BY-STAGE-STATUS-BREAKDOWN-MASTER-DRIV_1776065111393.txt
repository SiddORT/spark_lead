# 🔹 BACKEND LOGIC

---

## Step 1: Fetch Pipeline Master

* Get all stages
* Get all statuses under each stage
* Get colour for each status

---

## Step 2: Map Leads

For each lead:

* Identify:

  * Stage
  * Status
* Aggregate:

  * Count
  * Total deal value

---

## Step 3: Ensure FULL COVERAGE

Even if a status has:

* 0 leads
  → It MUST still appear in chart (with value 0)

---

# 📊 CHART IMPLEMENTATION

---

## 🔹 Use Stacked Bar Chart

Each Stage = 1 bar
Each Status = stacked segment

---

## 🔹 Dynamic Bars (IMPORTANT)

```tsx id="dynamic-bars"
const allStatuses = getStatusesFromPipelineMaster();

allStatuses.map((status) => (
  <Bar
    key={status.id}
    dataKey={status.name}
    stackId="a"
    fill={status.color}
  />
));
```

---

# 🔹 DATA TRANSFORMATION

```ts id="transform-master"
const formattedData = stagesFromMaster.map(stage => {
  const obj = { stage: stage.name };

  stage.statuses.forEach(status => {
    const data = getAggregatedData(stage.id, status.id);

    obj[status.name] = data.count || 0;
  });

  return obj;
});
```
