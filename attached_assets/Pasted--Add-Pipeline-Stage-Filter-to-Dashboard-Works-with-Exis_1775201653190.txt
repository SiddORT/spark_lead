## ✅ 3. Update Filtering Logic

Modify your lead filtering function:

```ts id="x7p4mn"
const filteredLeads = leads.filter((lead) => {
  const matchesSearch =
    !search ||
    lead.name.toLowerCase().includes(search.toLowerCase()) ||
    lead.company.toLowerCase().includes(search.toLowerCase());

  const matchesService =
    !selectedService || lead.service === selectedService;

  const matchesCompany =
    !selectedCompany || lead.company === selectedCompany;

  const matchesType =
    !selectedType || lead.type === selectedType;

  const matchesStage =
    !selectedStage || lead.stage === selectedStage;

  return (
    matchesSearch &&
    matchesService &&
    matchesCompany &&
    matchesType &&
    matchesStage
  );
});
```
