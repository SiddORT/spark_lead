## 📌 CURRENT SYSTEM BEHAVIOUR (REFERENCE)
1. Follow-up date is captured during **lead creation**.
2. It is shown in:
   - Lead Details → **Details Tab**
3. Follow-up tab logic:
   - If follow-up date == today → appears in **Due Today**
   - If follow-up date < today → appears as **Overdue**
4. This logic is **CORRECT and must remain unchanged**.

### 1. Move Follow-Up Date from Details Tab → Notes Tab
- Remove Follow-up Date field from:
  - ❌ Details Tab (UI only, do NOT delete data)
- Add Follow-up Date input in:
  - ✅ Notes Tab (while adding a note)
