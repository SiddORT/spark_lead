## 🚀 Move “Add New Service” Outside Dropdown (Better UX)

---

# 🎯 Objective

Improve UX of **Service selection in Create Lead form**:

### ❌ Current

* “+ Add New Service” inside dropdown
* Feels cluttered and hidden

### ✅ New Approach

* Add **“+ Add Service” button beside the Service field**
* Keep dropdown clean
* Open modal directly from button

---

# ✅ EXPECTED UI

```plaintext
SERVICE
[ Select service...             ]  [+ Add Service]
```

* Input = dropdown
* Button = inline (right side)

---

# 🧠 IMPLEMENTATION

---

## ✅ 1. Update Layout (Service Field + Button)

```tsx
<div className="flex items-center gap-2">
  
  <div className="flex-1">
    <CustomSelect
      value={selectedService}
      onChange={handleServiceChange}
      options={services}
      placeholder="Select service..."
    />
  </div>

  <button
    onClick={() => setShowServiceModal(true)}
    className="px-3 py-2 bg-teal-500/10 border border-teal-500/30 text-teal-400 rounded-lg hover:bg-teal-500/20 text-sm"
  >
    + Add
  </button>

</div>
```

---

## ✅ 2. Remove “Add New Service” from Dropdown

❌ Remove this:

```ts
{ label: "+ Add New Service", value: "__add_new__" }
```

---

## ✅ 3. Keep Modal Logic Same

```ts
const handleCreateService = async () => {
  const newService = await createService(...);

  const formattedService = {
    label: newService.name,
    value: newService.id,
  };

  // Update dropdown
  setServices((prev) => [...prev, formattedService]);

  // Auto-select new service
  setSelectedService(formattedService.value);

  setShowServiceModal(false);
};
```

---

## ✅ 4. Ensure Immediate UI Sync (CRITICAL)
