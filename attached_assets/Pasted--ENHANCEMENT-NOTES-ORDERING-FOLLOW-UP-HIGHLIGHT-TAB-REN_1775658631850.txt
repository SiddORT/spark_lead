---

## 🔹 CURRENT ISSUE

* Notes are not sorted properly
* Latest note is not always visible first

---

## 🔹 REQUIRED FIX

Sort notes in **descending order (latest first)**

---

## 🔹 IMPLEMENTATION

```ts

  (a, b) => new Date(b.createdAt) - new Date(a.createdAt)

```

* Older notes follow below

## 🔹 REQUIRED UX IMPROVEMENT

Make follow-up date:

* Visually distinct
* Clearly labeled
* Easily scannable

---

## 🔹 UI IMPLEMENTATION

```tsx

  <div>
    <p className="text-sm text-gray-300">{note.content}</p>
```
