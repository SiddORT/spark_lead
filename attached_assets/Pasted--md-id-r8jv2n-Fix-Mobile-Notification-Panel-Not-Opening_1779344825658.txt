# Fix Mobile Notification Panel Not Opening on Dashboard

The notification system is currently broken in mobile view.

When clicking the notification icon on the Dashboard page in mobile mode:
- the screen becomes blurred/dimmed
- but the actual notification panel/modal does NOT appear

This creates a broken and unusable mobile notification experience.

Screenshots have been reviewed for reference and understanding.

---

# CURRENT ISSUE

# MOBILE VIEW BEHAVIOUR

When user taps:

```txt
Notification Bell Icon
```

Current result:
- background overlay appears
- screen blurs/darkens
- interaction gets blocked

BUT:
- notification drawer/modal is not visible
- popup may be rendering off-screen
- z-index/layout is broken

---

# EXPECTED BEHAVIOUR

When user taps notification icon in mobile view:

The system should properly display:
- notification drawer
OR
- notification modal
OR
- slide-over panel

that is:
- fully visible
- accessible
- touch-friendly
- responsive

---

# POSSIBLE ROOT CAUSES

Investigate:

## 1. Z-Index Issues

The overlay may have:
- higher z-index than notification panel

Result:
- panel exists but hidden behind overlay

---

## 2. Off-Screen Positioning

The notification container may:
- still use desktop positioning

Example:
- `right: 20px`
- fixed width
- absolute positioning outside viewport

---

## 3. Overflow Hidden Issue

A parent container may be clipping:
- the notification drawer/modal

Check:
- `overflow: hidden`
- transform contexts
- stacking contexts

---

## 4. Responsive Breakpoint Logic Missing

Desktop dropdown logic may not be adapted for mobile.

---

# REQUIRED FIX

# IMPLEMENT PROPER MOBILE NOTIFICATION PANEL

Mobile notification UX must behave differently from desktop if required.

---

# REQUIRED MOBILE BEHAVIOUR

On mobile:
- notification panel should slide from:
  - top

Preferred modern SaaS behavior.

```txt
Mobile Notification Drawer
```

with:
- full visibility
- proper scrolling
- close button
- touch-friendly spacing

- maintain readable sizing
