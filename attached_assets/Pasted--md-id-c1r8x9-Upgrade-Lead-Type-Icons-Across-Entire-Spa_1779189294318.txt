# Upgrade Lead Type Icons Across Entire SparkLead System

The current icons used for Lead Types (`Hot`, `Warm`, `Cold`, `Ghosted`) look:
- immature
- emoji-like
- visually inconsistent
- out of place
- not aligned with the premium UI design language

The current icons reduce the professional feel of the CRM.

We need to completely redesign and replace the Lead Type icons across the ENTIRE system with a much more:
- sophisticated
- minimal
- enterprise-grade
- classy
- modern SaaS style

---

# IMPORTANT

This update must be implemented globally across the entire platform.

The icons should be updated everywhere including:
- Lead table
- Filters
- Dropdowns
- Kanban board
- Lead cards
- Details pages
- Badges
- Mobile views
- Notifications
- Search/filter chips
- Reports/analytics
- Any future reusable components

DO NOT update only one screen.

---

# CURRENT PROBLEM

Current icons:
- 🔥 Hot
- 🌞 Warm
- 🧊 Cold
- 👻 Ghosted

These icons:
- feel childish
- look like emojis
- clash with the modern UI
- do not match sidebar/dashboard icon styling
- reduce enterprise quality perception

---

# REQUIRED NEW DESIGN STYLE

The new icons must:
- match Lucide / Heroicons / modern SaaS iconography
- use clean vector-style outlines
- have balanced stroke width
- look subtle and elegant
- blend naturally with the rest of the UI
- feel premium and enterprise-ready

---

# RECOMMENDED ICON STYLE

Use icons from:
- Lucide React
- Heroicons
- Tabler Icons

Preferred icon style:
- outline or duotone
- 1.75px–2px stroke
- rounded edges
- minimal geometry
- modern SaaS aesthetic
