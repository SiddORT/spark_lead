````md id="d7m5k1"
```text

# REQUIRED COMPONENT LOGIC

Implement proper theme-aware logo rendering.

Example:

```tsx
const logoSrc = isDarkMode
  ? "/assets/branding/ort-logo-dark.png"
  : "/assets/branding/ort-logo-light.png";
```

Then:

```tsx
<img
  src={logoSrc}
  alt="ORT"
  className="ort-logo"
/>
```
````
