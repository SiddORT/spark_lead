# ✅ PART 2: SORT LEADS (LATEST FIRST)

---

## ✅ 6. Backend Sorting (RECOMMENDED)

```ts
const { data } = await supabase
  .from("leads")
  .select("*")
  .order("created_at", { ascending: false }); // 🔥 latest first
```

---

## ✅ 7. Frontend Sorting (MANDATORY FALLBACK)

```ts
const sortedLeads = [...leads].sort(
  (a, b) => new Date(b.created_at) - new Date(a.created_at)
);
```

---

## ✅ 8. Apply Sorting After Filtering

```ts
const finalLeads = filteredLeads.sort(
  (a, b) => new Date(b.created_at) - new Date(a.created_at)
);
```

---

## ✅ 9. Ensure New Lead Appears at Top

```ts
setLeads((prev) => [newLead, ...prev]);
```

---

❌ DO NOT:

```ts
[...prev, newLead] // wrong order
```
