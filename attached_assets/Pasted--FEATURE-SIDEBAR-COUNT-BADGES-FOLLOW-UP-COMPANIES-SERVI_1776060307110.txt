# 🚀 FEATURE: SIDEBAR COUNT BADGES (FOLLOW UP, COMPANIES, SERVICES)

---

# 🎯 OBJECTIVE

Display dynamic count badges in the left sidebar for:

* 📌 **Follow Up** → Number of leads in Follow-Up tab
* 🏢 **Companies** → Total companies count
* 🛠 **Services** → Total services count

These counts should be:
✔ Real-time / near real-time
✔ Accurate
✔ Visible from all pages
✔ UI consistent with design

---

# ✅ EXPECTED UI

---

### 📍 SIDEBAR VIEW

```
Follow Up        [ 8 ]
Companies        [ 24 ]
Services         [ 12 ]
```

* Badge should be:

  * Rounded (pill style)
  * Subtle glow / theme aligned
  * Right aligned
  * Not breaking layout

---

# 🧠 LOGIC DEFINITION

---

## 🔹 FOLLOW UP COUNT

Count =

* Leads where:

  * followUpDate = today
    OR
  * followUpDate < today (overdue)

```ts
followUpCount = leads.filter(
  lead => isToday(lead.followUpDate) || isPast(lead.followUpDate)
).length;
```

---

## 🔹 COMPANIES COUNT

```ts
companiesCount = totalCompanies.length;
```

---

## 🔹 SERVICES COUNT

```ts
servicesCount = totalServices.length;
```

# 🏗️ BACKEND (IF REQUIRED)

---

## 🔹 API ENDPOINT (OPTIONAL OPTIMIZATION)

```ts
GET /dashboard/sidebar-counts

Response:
{
  followUp: number,
  companies: number,
  services: number
}
```

✔ Improves performance

# 🎨 FRONTEND IMPLEMENTATION

---

## 🔹 STEP 1: CREATE BADGE COMPONENT

```tsx
const SidebarBadge = ({ count }) => {
  if (!count || count === 0) return null;

  return (
    <span className="ml-auto px-2 py-0.5 text-xs rounded-full bg-teal-500/10 text-teal-400">
      {count}
    </span>
  );
};
```

---

## 🔹 STEP 2: APPLY TO SIDEBAR ITEMS

```tsx
<SidebarItem label="Follow Up">
  <SidebarBadge count={followUpCount} />
</SidebarItem>

<SidebarItem label="Companies">
  <SidebarBadge count={companiesCount} />
</SidebarItem>

<SidebarItem label="Services">
  <SidebarBadge count={servicesCount} />
</SidebarItem>
```

---

## 🔹 STEP 3: GLOBAL STATE (IMPORTANT)

Use centralized state (Redux / Context / React Query)

```ts
const { followUpCount, companiesCount, servicesCount } = useSidebarCounts();
```

---

## 🔹 STEP 4: AUTO REFRESH

Update counts when:

* Lead created / updated
* Follow-up date changed
* Company added
* Service added

```ts
useEffect(() => {
  refetchCounts();
}, [leadChanges, companyChanges, serviceChanges]);
```
