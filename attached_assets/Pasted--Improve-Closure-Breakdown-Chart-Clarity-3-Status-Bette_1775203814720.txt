## ✅ 1. Prepare Data (From Leads)

```ts id="9xq2kf"
const wonCount = leads.filter(
  (l) => l.stage === "Closure" && l.status === "Won"
).length;

const lostCount = leads.filter(
  (l) => l.stage === "Closure" && l.status === "Lost"
).length;

const postponedCount = leads.filter(
  (l) => l.stage === "Closure" && l.status === "Postponed"
).length;
```

---

## ✅ 2. Create Chart Data

```ts id="m7f2kd"
const closureData = [
  { name: "Won", value: wonCount, color: "#22c55e" },      // green
  { name: "Lost", value: lostCount, color: "#ef4444" },    // red
  { name: "Postponed", value: postponedCount, color: "#f59e0b" }, // yellow
];
```

---

## ✅ 3. Render Pie / Donut Chart (Recharts Example)

```tsx id="3p9k2v"
<PieChart width={250} height={250}>
  <Pie
    data={closureData}
    dataKey="value"
    nameKey="name"
    cx="50%"
    cy="50%"
    outerRadius={90}
    innerRadius={50}
  >
    {closureData.map((entry, index) => (
      <Cell key={index} fill={entry.color} />
    ))}
  </Pie>

  <Tooltip
    contentStyle={{
      backgroundColor: "#0B0F19",
      border: "1px solid #333",
      color: "#fff",
      borderRadius: "8px",
    }}
    labelStyle={{ color: "#fff" }}
    itemStyle={{ color: "#fff" }}
  />
</PieChart>
```

---

## ✅ 4. Add Legend (IMPORTANT)

```tsx id="f6z1lm"
<div className="mt-4 flex justify-center gap-4 text-sm">
  {closureData.map((item) => (

      <span
        className="w-3 h-3 rounded-full"
        style={{ backgroundColor: item.color }}
      />
      <span className="text-gray-300">
        {item.name} ({item.value})
      </span>
    </div>
  ))}
</div>
```
